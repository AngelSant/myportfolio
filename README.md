# Características
- Diseño responsive para escritorio, tablet y dispositivos móviles.
- Modo claro y modo oscuro.
- Persistencia de preferencia visual mediante localStorage.
- Navegación con indicación de ruta activa.
- Iconos asociados a cada sección.
- Animación inicial de bienvenida.
- Transiciones de entrada al cambiar de página.
- Tarjetas reutilizables para proyectos, servicios y certificaciones.
- Despliegue automático en GitHub Pages mediante GitHub Actions.
- Exportación estática compatible con GitHub Pages.
